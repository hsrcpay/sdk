# @hsrcpay/sdk

HSRCPAY API'leri için resmi Node.js SDK paketidir.

## Amaç

Node.js ekosisteminde HSRCPAY entegrasyonunu hızlı, tutarlı ve sürdürülebilir hale getirmek.

## Kapsam

- Paket adı: `@hsrcpay/sdk`
- Dağıtım hedefi: `dist/` üzerinden JavaScript çıktı
- Tüketim modeli: uygulama servisleri, backend API katmanları, server-side iş akışları

## Durum

Bu dizin şu an SDK altyapı şablonunu içerir. Üretim API yüzeyi aşamalı olarak genişletilecektir.
