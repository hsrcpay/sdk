# HSRCPAY Go SDK

Go tabanlı uygulamalar için HSRCPAY resmi SDK alanıdır.

## Amaç

Kurumsal backend servislerinin ödeme orkestrasyon akışlarını Go ile güvenli ve yalın biçimde yönetmesini sağlamak.

## Kapsam

- Go modül yapısı
- HSRCPAY API istemci katmanı
- Üretim senaryolarına uygun hata, yeniden deneme ve yapılandırma standartları

## Durum

Bu klasör şablon aşamasındadır. API yüzeyi ve modül yapısı resmi dokümantasyonla uyumlu olarak devreye alınacaktır.

## Lisans

Bu SDK kök dizindeki `LICENSE` dosyasına tabidir.
