# HSRCPAY Kotlin SDK

Kotlin tabanlı uygulamalar için HSRCPAY resmi SDK alanıdır.

## Amaç

Kotlin geliştiricilerine modern, okunabilir ve üretim odaklı bir ödeme API istemci deneyimi sunmak.

## Kapsam

- Kotlin library yapısı
- JVM ve Kotlin-first kullanım senaryolarına uyum
- Genişletilebilir istemci tasarımı ve güvenli yapılandırma yaklaşımı

## Durum

Bu klasör şablon aşamasındadır. API katmanı ve yayın hattı, resmi ürün yol haritasına göre eklenecektir.

## Lisans

Bu SDK kök dizindeki `LICENSE` dosyasına tabidir.
