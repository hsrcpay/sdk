# HSRCPAY Java SDK

Java ekosistemi için HSRCPAY resmi SDK alanıdır.

## Amaç

Kurumsal JVM projelerinde HSRCPAY entegrasyonunu standartlaştırmak ve bakım maliyetini azaltmak.

## Kapsam

- Java library yapısı
- Kurumsal servis mimarilerine uygun istemci arayüzü
- Sürüm uyumluluğu ve geriye dönük stabilite odaklı yaklaşım

## Durum

Bu klasör şablon aşamasındadır. Paketleme, artefact dağıtımı ve entegrasyon dokümanları sonraki fazda eklenecektir.
