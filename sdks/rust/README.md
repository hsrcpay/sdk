# HSRCPAY Rust SDK

Rust tabanlı sistemler için HSRCPAY resmi SDK alanıdır.

## Amaç

Yüksek performans ve güvenilirlik gerektiren ödeme akışlarında Rust geliştiricilerine güçlü bir istemci katmanı sunmak.

## Kapsam

- Rust crate yapısı
- Tip güvenli API istemci yaklaşımı
- Performans odaklı, hataya dayanıklı entegrasyon modeli

## Durum

Bu klasör şablon aşamasındadır. Stabil API sözleşmesi ve crate sürümleme politikası ilerleyen fazda yayınlanacaktır.

## Lisans

Bu SDK kök dizindeki `LICENSE` dosyasına tabidir.
