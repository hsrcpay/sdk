## Bu Deponun Amacı

Bu depo, HSRCPAY API client yaklaşımına uygun şekilde:

- Çoklu dil SDK stratejisini tek merkezden yönetmek,
- Ortak standartları ve varlıkları bir arada tutmak,
- Kurumsal entegrasyon ekipleri için tutarlı bir temel sağlamak için hazırlanmıştır.

Şu aşamada depo, bilinçli olarak şablon odaklıdır ve teknik uygulama detaylarını minimum seviyede tutar.
